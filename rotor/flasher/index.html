Overhead Companion Rotor — Web Flasher

alt/az pointer that slews to whatever the Overhead dashboard is tracking

Flash the default build (plain ESP32 + 28BYJ-48 / ULN2003 steppers) straight from your browser over USB. Plug the ESP32 in, click Install, and choose its serial port.

Your browser can't do Web Serial — use desktop Chrome or Edge. Serial access was blocked — allow it and retry.

Requires a Chromium browser (Chrome / Edge) served over HTTPS or localhost. Web Serial does not work from a file:// page or over plain HTTP.

NEMA 17 / custom builds: this flasher only ships the default ESP32 + 28BYJ preset. Different drivers, pins, or gear ratios build from source instead — edit rotor/src/config.h (the NEMA 17 + A4988/TMC2209 preset is in there) and run pio run -d rotor -t upload.

After flashing, open a serial monitor at 115200 and calibrate: CAL EL, CAL AZ, SETNORTH — no gear-ratio math needed. See rotor/README.md.
